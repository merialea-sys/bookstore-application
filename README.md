# Book Store Application

## Overview
This project is a backend focused application designed to demonstrate object-oriented programming principles and database management using SQLite. It allows users to manage author records, add books with associated author IDs and retrieve related data.

## Features
- Manages authors and books tables with a one to many relationship
- Uses python classes to relate to the database
- Populates the database with sample data for quick testing

## Technologies Used
* Python
* SQLite3

## Installation and Setup
1. Clone the repository:
   git clone (Git Repo)[https://github.com/merialea-sys/bookstore-application]
   cd bookstore-application
2. Setup vitual environment
   run pipenv install
   run pipenv shell

## Database initialization
python lib/seed.py

## Running the application
You can interact with the database using Python scripts
python lib/cli.py
